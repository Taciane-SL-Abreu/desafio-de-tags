Curiosidades sobre a estrutura do HTML

O que são tags?

Tags são marcações do HTML* utilizadas para estruturar websites.

Existe vários tipos de tags como, por exemplo, para deixar o texto em negrito, sublinhado, destacado e até mesmo criar paragráfos como este que você está lendo.

* Tradução: Linguagem de Marcação de Hipertexto.

Exemplos de tags:

  • H1 até H6: Níveis de títulos, H1 como o maior e o H6 como o menor.
  • I: O texto fica em ítalico.
  • P: Para criar um paragráfo.
Como funcionam?

A maioria das tags são simbolizadas pelos sinais de maior que: < e menor que: >, e barra: /.

Sendo <p> utilizando para definir abertura de tag e </p> como fechamento [está é uma representação da tag P para criação de um paragráfo].

Algumas tags obsoletas:

Assim como todas as tecnologias, o HTML também está em constante mudança. Atualmente, estamos na versão HTML5 e, por isso, algumas tags deixam de ser usadas e outras são acrescentadas. Exemplos:

  1. <font>, ela era usada para estilizar fontes, cor e tamanho das letras. Porém ela foi descontinuada e a estilização das páginas HTML são feitas via CSS*.
  2. <acronym>, ela permitia à autores claramente indicar que uma seqüência de caracteres compõe um acrônimo ou uma abreviação de uma palavra. Porém no HTML5 ela foi substituída pela <abbr>.
  3. <image>, ela adicionava uma imagem a página, mas foi trocada pela tag <img>, na versão HTML5.

* Tradução: Folhas de Estilo em Cascata.
